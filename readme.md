## 🛠️ Development Setup

This setup is common across all weeks and projects.

### Requirements

* Rust
* Solana CLI
* Anchor
* Node.js & Yarn

### Basic Commands

```bash
# Set local network
solana config set --url localhost

# Create wallet
solana-keygen new

# Start local validator
solana-test-validator

# Anchor project (example: week-0)
anchor init week-0
cd week-0

# Build & test
anchor build
anchor test
```
